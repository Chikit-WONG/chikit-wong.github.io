# Chi Kit Wong — Academic Homepage

A lightweight, responsive academic homepage designed for GitHub Pages.

## Publish as a personal GitHub Pages site

1. Create a **public** repository named exactly `chikit-wong.github.io`.
2. Upload all files in this folder to the repository root.
3. Open **Settings → Pages** in the repository.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. The site will be available at `https://chikit-wong.github.io/` after deployment.

## Files

- `index.html` — page content
- `style.css` — responsive light/dark design
- `script.js` — theme switcher and subtle entrance animations
- `.nojekyll` — serves the files directly without Jekyll processing

## Customize

Edit the text in `index.html`. Replace the GitHub avatar URL with a local portrait later if desired.
