# Chi Kit Wong — Academic Homepage

Personal academic homepage hosted with GitHub Pages.

## Deploy

Upload all files in this folder to the root of the `chikit-wong.github.io` repository. GitHub Pages should use the `main` branch and `/ (root)` folder.

## Search indexing

The site includes a canonical URL, Person structured data, `robots.txt`, and `sitemap.xml`.

After deployment, add `https://chikit-wong.github.io/` as a URL-prefix property in Google Search Console, verify ownership, submit `sitemap.xml`, and use URL Inspection to request indexing.
